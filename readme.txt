== Description ==
Add Webtrends Analytics tag to your website. Allows for set-up and configuration.
This plugin requires that you have access to an account from Webtrends.

This plugin is not a Webtrends product.
This plugin is not supported by Webtrends.

== Installation ==
1. Upload the plugin directory and all the included files to the /wp-content/plugins/ folder.
2. Activate the plugin through the Plugins menu in WordPress.
3. Access the Webtrends options page and configure the plugin settings.
